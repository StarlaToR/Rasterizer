# Contrôle

## Déplacer la caméra : 
- Droite : 'Flèche droite'
- Gauche : 'Flèche gauche'
- Haut : 'Flèche du haut'
- Bas : 'Flèche du bas'

## Pivoter : 
- Axe X : 'S' (positif),'W' (négatif)
- Axe Y : 'S' (positif),'W' (négatif)
